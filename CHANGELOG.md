# Changelog

## [0.0.4] - 2025-12-10

- Add MultiAgent support to OxideAgent's core
- Add multiAgent sections to TUI to allow us to swap between sessions and see active sessions

## [0.0.3] - 2025-12-09

- Seperated Agents, models and sessions to be individual things so we can update the history, system prompt or active model easily

## [0.0.2] - 2025-10-13

- Added MCP support both via url and via commands (docker, uvx, npx)
- Fixed all clippy errors in src and tests

## [0.0.1] - 2025-09-30

- Initial release with unified config system (JSON/YAML/TOML support)
- Added CLI config option with precedence over config files
- Implemented CI pipeline with tarpaulin coverage
